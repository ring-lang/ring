==============
The Cards Game
==============

Developed using Ring and RingQt.

(1) Each player get 5 cards (Or More!)
(2) The cards are unknown to any one.
(3) Each time one player click on one card to see it.
	3.1 if the card is identical to another card the play get a point for each card.
	3.2 if the card value is “5” the player get points for all visible cards.
(4) To win get more score than the other player.
