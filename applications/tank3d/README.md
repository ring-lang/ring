Tank3D Game
================

Developed using Ring and RingRayLib

The source code is 100% generated using Claude Code (Opus 4.6)

# Using RingPM

To quickly install and run the game using the Ring Package Manager

	ringpm install tank3d from ringpackages
	ringpm run tank3d
