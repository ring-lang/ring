Stock Analysis 
==============

This application is a stock analysis and simulation tool that fetches historical price data 
from Yahoo Finance, parses it into usable time series, calculates momentum-based returns, and 
simulates portfolio performance against a benchmark (QQQ)

Developed using Ring, RingQt, RingLibCurl & RingJSONLib

Author: Azzeddine Remmal
