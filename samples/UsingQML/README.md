UsingQML Folder
===============

In this folder we store the samples related to using Ring and QML.

* QML_SomeFeatures folder: Samples about some of QML features (Animations, Charts, 3D, etc.)

* Android_WebView folder: Using QML WebView control in our Qt Widgets apps for Android

* RingQML_Communication folder: How to call QML from Ring and call Ring from QML

The previous samples uses QQuickWidget (i.e. using QML in Qt Widgets app)

* WebAsm_ChatInterface folder: Using QQuickView and developing an app using Ring and QML fro Desktop/Android and WebAssembly
